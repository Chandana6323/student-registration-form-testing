# Student Registration Form Testing

## Project Description
This project demonstrates manual testing of a simple Student Registration Form created using HTML.

## Project Files
- student_registration_form.html – HTML form with validation
- Student_Registration_Test_Cases.xlsx – Test cases written for manual testing

## Test Scenarios Covered
- Valid registration
- Empty field validation
- Invalid phone number validation
- Form validation messages

## Tools Used
- HTML
- Excel
- Manual Testing
